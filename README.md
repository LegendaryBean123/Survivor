# Survivor

Game made using pygame. Survival game where you try to kill as many enemies as possible to up your score.

## Download & Installation

### Prerequisites
- Python 3.7 or higher
- pip (Python package manager)

### Steps to Download and Install
#### Recommended:
Download the .exe file from the dist folder
Then just click and play
#### Alternet methods:

1. **Clone or Download the Repository**
   ```bash
   git clone <repository-url>
   cd Survivor
   ```
   Or download as ZIP and extract to your desired location.

2. **Install Required Dependencies**
   ```bash
   pip install pygame
   ```

3. **Run the Game**
   ```bash
   python game.py
   ```

## How to Play

- **Objective**: Survive as long as possible while defeating enemies to increase your score.
- **Controls**: 
  - Use **arrow keys** or **WASD** to move your character
  - Use **mouse** to aim and click to shoot
- **Strategy**: Avoid enemy attacks while defeating as many enemies as possible to rack up points using the terrain to your advantage.
- **Gameplay**: Your score increases with each enemy defeated. How long can you survive?

## Game Features

- Dynamic enemy spawning
- Tile-based map system
- Multiple enemy types (bats, blobs, skeletons)
- Score tracking
- Smooth character movement and shooting mechanics
